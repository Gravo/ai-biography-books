# 第一章：深度学习教父

## 1.1 从巴黎到纽约

Yann LeCun（杨立昆），1960年出生于法国巴黎郊区。他是卷积神经网络（CNN）的发明者、深度学习三巨头之一、2018年图灵奖得主。

**关键里程碑：**

| 年份 | 事件 |
|------|------|
| 1960 | 出生于法国巴黎 |
| 1987-1988 | 在AT&T贝尔实验室开发LeNet，首个成功的手写数字识别系统 |
| 1998 | 发表开创性论文《Gradient-Based Learning Applied to Document Recognition》 |
| 2003 | 加入纽约大学，任计算机科学教授 |
| 2013 | 加入Facebook（现Meta），任首席AI科学家 |
| 2018 | 与Hinton、Bengio共同获得图灵奖 |
| 2025.11 | 离开Meta，创办AMI Labs |
| 2026.3 | AMI Labs完成10.3亿美元种子轮融资 |

## 1.2 卷积神经网络之父

LeCun最著名的贡献是卷积神经网络（CNN）：

- **LeNet-5**（1998）：首个成功的CNN架构，用于手写数字识别
- **反向传播算法**：与Hinton等人共同推广
- **深度学习基础**：奠定了现代AI视觉技术的基础

**影响领域：**
- 图像识别
- 自动驾驶视觉系统
- 医学影像分析
- 人脸识别

# 第二章：LLM的四大致命缺陷

## 2.1 2025达沃斯预言

2025年冬季达沃斯"技术辩论"现场，LeCun抛出震撼观点：

> **"当前的大语言模型（LLM）范式将在3-5年内被淘汰。"**

他直言：

> **"LLM就像从未接触过物理世界的'数字婴儿'。"**

尽管LLM能处理海量文本，却无法理解"两个碰撞的球体必须遵守动量守恒定律"这类基础物理规则。

## 2.2 四大核心缺陷

### 缺陷一：缺乏对物理世界的理解

LLM通过文本学习，从未真正"看见"或"触摸"过世界。它们知道"球会掉落"这个词组，却不理解重力是什么。

**示例：**
- GPT能描述"扔球"的物理过程
- 但无法可靠控制机器人抓取真实物体

### 缺陷二：缺乏持久记忆

尽管GPT-4 Turbo支持128k tokens上下文，但本质仍是"滑动窗口式记忆"。

**问题：**
- 无法真正"记住"过去的对话
- 每次交互都是新的开始
- 无法像人类一样积累经验

### 缺陷三：缺乏推理能力

**LeCun的名言：**

> **"LLM擅长操控语言，但并不擅长思考。"**

LLM在回答问题时，往往只是依据已有数据进行拼凑，缺乏对问题本质的真正理解。

### 缺陷四：缺乏复杂规划能力

LLM无法进行多步骤、长期规划：

- **人类**：能规划"去超市买菜"的完整流程（出门→交通→选购→结账→回家）
- **LLM**：只能逐词生成，没有"目标→子目标→行动"的层次化规划

# 第三章：世界模型愿景

## 3.1 什么是世界模型？

**世界模型（World Model）**是LeCun提出的下一代AI核心概念：

> 一个能学习并理解物理世界结构与规律的AI系统，能预测行为的后果并规划行动。

**关键能力：**

| 能力 | 说明 |
|------|------|
| **物理理解** | 懂重力、懂因果、懂空间关系 |
| **预测能力** | 能模拟"如果这样做，会发生什么" |
| **规划能力** | 能制定多步骤行动计划 |
| **持久记忆** | 能积累经验并复用 |

## 3.2 JEPA架构

**联合嵌入预测架构（Joint Embedding Predictive Architecture, JEPA）**是LeCun提出的技术方案：

**核心思想：**
- 不预测下一个token，而是预测抽象表示
- 通过观察视频和与环境交互来学习世界如何运行
- 分层结构支持更抽象、更长期的预测

**版本演进：**

| 模型 | 时间 | 特点 |
|------|------|------|
| **I-JEPA** | 2023.6 | 图像联合嵌入预测，首个基于世界模型愿景的AI模型 |
| **V-JEPA** | 2024 | 视频联合嵌入预测 |
| **V-JEPA 2** | 2025.6 | "现实的抽象数字孪生"，比英伟达Cosmos快30倍 |

**LeCun对V-JEPA 2的描述：**

> "我们相信，世界模型将开启机器人技术的新纪元，使现实世界中的AI主体能够帮助完成家务和体力任务，而无需海量的机器人训练数据。"

# 第四章：离开Meta

## 4.1 离职背景

2025年11月19日，LeCun宣布将在年底离开效力十年的Meta，创办自己的AI公司。

**离职原因：**

1. **技术路线分歧**：LeCun长期主张LLM无法实现人类水平智能，但Meta的LLaMA系列正是LLM产品
2. **商业化冲突**：学术路线与公司"研究直接服务产品"的转型方向冲突
3. **独立愿景**：想更快、更便宜、更好地实现世界模型

**LeCun离开时的话：**

> **"这玩意儿我自己干，能更快更便宜更好。"**

## 4.2 与扎克伯格的关系

尽管离职，LeCun与扎克伯格保持良好关系：

- 离职公告配图是与扎克伯格的合影
- Meta表示将与AMI Labs保持合作
- AMI Labs走开源路线，与Meta理念一致

# 第七章：观点与争议

## 7.1 对LLM的持续批判

**核心论点：**

1. **LLM无法超越训练数据**
   - 无法创造新概念
   - 面对复杂现实问题表现乏力

2. **自回归生成的局限**
   - 逐词生成导致缺乏全局规划
   - 无法保证逻辑一致性

3. **"文本→智能"的死胡同**
   - 文本只是世界的符号表示
   - 真正的智能需要理解物理世界

## 7.2 与英伟达的辩论

**2025年GTC大会**：LeCun与英伟达首席科学家Bill Dally就"类人AI发展路径"展开辩论

**LeCun观点：**
- 类人AI比你想象的更近
- 但路径完全错了——不应依赖LLM

## 7.3 "六个月后"预言

**种子轮融资后的评论：**

> **"六个月后，每家公司都会自称是世界模型来筹集资金。"**

# 结语：一场豪赌的意义

LeCun的$1B豪赌，不只是资金，更是信念：

**他在赌：**
- LLM不是AI的终点
- 理解物理世界才是智能的核心
- 世界模型将在5年内取代LLM成为AI系统的核心

**如果赌对了：**
- 开启机器人技术新纪元
- AI能真正帮助人类处理物理世界的任务
- 欧洲AI产业翻盘

**如果赌错了：**
- 10亿美元投入可能打水漂
- 但科学探索本就需要勇气

正如LeCun所说：

> **"LLM能通过律师资格考试，但清不了一张餐桌。"**

他赌的是一个能真正"干活"的AI。
